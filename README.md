# Marisha Website

Personal website for Marisha Lakhiani.

## Files

- `index.html` - Main website file
- `HERO-IMAGE.jpg` - Hero section image

## Deployment

This site is deployed using GitHub Pages. See `DEPLOYMENT.md` for detailed instructions.

### Quick Deploy

1. Create a GitHub repository named `marisha-site`
2. Run `./push-to-github.sh`
3. Enable GitHub Pages in repository settings

## Local Development

Simply open `index.html` in a web browser to view the site locally.
